## 概述

PyTorch-NEAT 提供了三种不同类型的神经网络，它们都可以与NEAT算法结合使用，但在结构、能力和适用场景上各有特点。

> 注：从 `pytorch_neat` 子库能力看，这三种网络实现都存在。
> 但从当前仓库的工程接入状态看，`train.py` + Hydra + `Experiment` 训练框架目前只接入了 `RecurrentNet`。
> `AdaptiveLinearNet` 和 `AdaptiveNet` 仍主要作为底层库实现与测试对象保留。

### 核心区别

| 维度 | RecurrentNet | AdaptiveLinearNet | AdaptiveNet |
|------|--------------|-------------------|-------------|
| **拓扑结构** | 输入→隐藏→输出 | 输入→输出 | 输入→隐藏→输出 |
| **权重来源** | 直接编码（NEAT） | CPPN生成 | CPPN生成 |
| **自适应性** | ❌ 固定 | ✅ 动态更新 | ✅ 动态更新 |
| **记忆机制** | 隐藏层状态 | 权重变化 | 隐藏层状态+权重变化 |
| **计算复杂度** | 低 | 中 | 高 |
| **训练速度** | 快 | 中 | 慢 |

### 💡 选择建议

```
简单任务（CartPole等）
    → RecurrentNet
    - 训练最快
    - 实现简单
    - 足够解决问题

需要短期记忆（T-Maze等）
    → AdaptiveLinearNet
    - 自适应能力
    - 计算适中
    - 适合记忆任务

复杂任务（多智能体、长期规划等）
    → AdaptiveNet
    - 最强能力
    - 训练较慢
    - 适合研究

资源受限
    → RecurrentNet > AdaptiveLinearNet > AdaptiveNet

研究自适应机制
    → AdaptiveLinearNet 或 AdaptiveNet
```

## 详细说明

### RecurrentNet - 递归神经网络

#### 📝 基本信息

**文件位置**: `pytorch_neat/recurrent_net.py`

**核心思想**: 标准的递归神经网络实现，支持任意拓扑结构，权重直接从NEAT基因组中提取。

#### 🏗️ 网络结构

```
输入层 (n_inputs)
   ↓
   ├─────→ 隐藏层 (n_hidden) ↺ (循环连接)
   │          ↓
   └─────→ 输出层 (n_outputs) ↺ (循环连接)
```

支持的连接类型：
- 输入 → 隐藏
- 隐藏 → 隐藏（循环）
- 输出 → 隐藏（反馈）
- 输入 → 输出
- 隐藏 → 输出
- 输出 → 输出（循环）

#### 💻 代码示例

```python
from pytorch_neat.recurrent_net import RecurrentNet
from pytorch_neat.activations import tanh_activation

# 从基因组创建网络
net = RecurrentNet.create(
    genome,                      # NEAT基因组
    config,                      # NEAT配置
    batch_size=4,                # 批量大小
    activation=tanh_activation,  # 激活函数
    n_internal_steps=1,          # 内部迭代次数
    use_current_activs=False     # 是否使用当前激活值
)

# 重置网络状态
net.reset(batch_size=4)

# 前向传播
for step in range(max_steps):
    outputs = net.activate(states)  # states: (batch_size, n_inputs)
    # outputs: (batch_size, n_outputs)
```

#### 🔑 关键参数

**n_internal_steps** (内部迭代次数)
- 默认值: 1
- 作用: 隐藏层在单个时间步内迭代的次数
- 效果: 增加可以提升计算能力，但会变慢
- 推荐: 简单任务用1，复杂任务用2-5

**use_current_activs** (使用当前激活)
- 默认值: False
- 作用: 计算输出时是否使用当前时刻的隐藏层激活
- False: 使用上一时刻的隐藏层（标准RNN）
- True: 使用当前时刻的隐藏层（更快的信息流）

**activation** (激活函数)
- 默认值: sigmoid_activation
- 推荐: sigmoid（输出有界）、tanh（零中心）、relu（稀疏激活）

#### ⚙️ 工作原理

1. **初始化**: 从基因组提取所有连接和权重，构建6个连接矩阵
2. **前向传播**:
   ```python
   # 更新隐藏层（迭代n_internal_steps次）
   hidden(t) = activation(response * (
       W_ih @ input(t) +
       W_hh @ hidden(t-1) +
       W_oh @ output(t-1)) + bias_h)
   
   # 更新输出层
   output(t) = activation(response * (
       W_io @ input(t) +
       W_ho @ hidden(t) +
       W_oo @ output(t-1)) + bias_o)
   ```
3. **状态保存**: 隐藏层和输出层的激活值保存到下一时间步

#### ✅ 优点

- **简单直观**: 就是标准的RNN，易于理解和调试
- **训练快速**: 无CPPN开销，评估速度最快
- **内存占用小**: 只存储权重矩阵，无额外计算图
- **稳定可靠**: 权重固定，行为可预测

#### ❌ 缺点

- **无自适应**: 权重固定，无法在episode内学习
- **搜索空间大**: 所有权重都要进化，参数量大
- **无几何规律**: 无法利用空间结构信息

#### 🎯 适用场景

- ✅ 简单控制任务（CartPole、Pendulum、LunarLander）
- ✅ 不需要在线学习的任务
- ✅ 需要快速评估的场景（大规模实验）
- ✅ 作为baseline比较
- ✅ 教学和学习NEAT

### AdaptiveLinearNet - 自适应线性网络

#### 📝 基本信息

**文件位置**: `pytorch_neat/adaptive_linear_net.py`

**核心思想**: 无隐藏层的自适应网络，权重可以根据输入输出的激活值动态更新，实现快速适应和短期记忆。

#### 🏗️ 网络结构

```
输入层 (n_inputs)
   ↓
   W(t) - 自适应权重矩阵
   ↓
输出层 (n_outputs)

权重更新:
W(t+1) = W(t) + CPPN(坐标, input(t), output(t), W(t))
```

特点：
- **无隐藏层**: 直接连接输入和输出
- **自适应权重**: 每个时间步根据激活值更新权重
- **空间编码**: 使用2D坐标表示神经元位置
- **稀疏连接**: 只更新初始非零的连接

#### 💻 代码示例

```python
from pytorch_neat.adaptive_linear_net import AdaptiveLinearNet
from pytorch_neat.activations import tanh_activation, identity_activation

# 定义神经元空间坐标
input_coords = [
    [-1.0, 0.0],  # 左侧传感器
    [0.0, 0.0],   # 前方传感器
    [1.0, 0.0],   # 右侧传感器
    [0.0, -1.0]   # 额外输入（如颜色）
]
output_coords = [
    [-1.0, 0.0],  # 左转
    [0.0, 0.0],   # 前进
    [1.0, 0.0]    # 右转
]

# 从基因组创建网络
net = AdaptiveLinearNet.create(
    genome,
    config,
    input_coords=input_coords,
    output_coords=output_coords,
    weight_threshold=0.4,           # 权重阈值（稀疏化）
    weight_max=3.0,                 # 权重最大值
    activation=tanh_activation,     # 输出激活函数
    cppn_activation=identity_activation,  # CPPN输出激活
    batch_size=4,
    device='cpu'
)

# 重置网络
net.reset()

# 前向传播（自动更新权重）
for step in range(max_steps):
    outputs = net.activate(states)
    # 权重已自动更新，适应当前情况
```

#### 🔑 关键参数

**input_coords / output_coords** (神经元坐标)
- 类型: List[List[float]]
- 格式: [[x1, y1], [x2, y2], ...]
- 作用: 定义神经元在2D空间中的位置
- 设计建议:
  - 根据任务语义排列（如传感器的物理位置）
  - 对称布局可能产生对称行为
  - 范围通常在[-1, 1]之间

**weight_threshold** (权重阈值)
- 默认值: 0.2
- 作用: 初始权重绝对值小于此值的被设为0
- 效果: 控制网络稀疏性
- 推荐: 0.2-0.5，值越大网络越稀疏

**weight_max** (权重最大值)
- 默认值: 3.0
- 作用: 权重被限制在[-weight_max, weight_max]
- 效果: 防止权重爆炸
- 推荐: 2.0-5.0

**cppn_activation** (CPPN输出激活)
- 默认值: identity_activation
- 作用: 对CPPN生成的权重更新应用激活函数
- 选项:
  - identity: 不限制（权重更新可能很大）
  - tanh: 限制在(-1, 1)（更稳定）
  - sigmoid: 限制在(0, 1)（只增加权重）

#### ⚙️ 工作原理

1. **初始化**: 使用CPPN根据坐标生成初始权重矩阵
   ```python
   W(0) = CPPN(x_in, y_in, x_out, y_out, 0, 0, 0)
   ```

2. **前向传播**: 计算输出
   ```python
   output(t) = activation(W(t) @ input(t))
   ```

3. **权重更新**: 使用CPPN计算更新量
   ```python
   delta_w = CPPN(x_in, y_in, x_out, y_out, 
                  input(t), output(t), W(t))
   W(t+1)[expressed] = W(t)[expressed] + delta_w[expressed]
   ```

4. **稀疏性保持**: 只在初始非零的连接上更新

#### ✅ 优点

- **自适应能力**: 可以在episode内学习
- **短期记忆**: 通过权重变化存储信息
- **计算高效**: 无隐藏层，计算快
- **参数效率**: CPPN间接编码，参数少
- **空间规律**: 利用几何结构

#### ❌ 缺点

- **表达能力有限**: 无隐藏层，无法表示复杂函数
- **无长期记忆**: 无循环连接，记忆能力有限
- **坐标依赖**: 需要合理设计神经元坐标
- **稀疏限制**: 网络结构在初始化后固定

#### 🎯 适用场景

- ✅ 需要短期记忆的任务（T-Maze、延迟匹配）
- ✅ 需要快速适应的任务（变化的环境）
- ✅ 输入输出维度较小的任务
- ✅ 研究Hebbian学习和元学习
- ✅ 资源受限但需要自适应

### AdaptiveNet - 自适应递归网络

#### 📝 基本信息

**文件位置**: `pytorch_neat/adaptive_net.py`

**核心思想**: 结合递归结构和自适应权重，是最复杂和最强大的网络类型。包含隐藏层和循环连接，其中隐藏层的循环权重可以动态更新。

#### 🏗️ 网络结构

```
输入层 (n_inputs)
   ↓
   W_ih - 固定权重
   ↓
隐藏层 (n_hidden) ↺ W_hh(t) - 自适应循环权重
   ↓
   W_ho - 固定权重
   ↓
输出层 (n_outputs)

权重更新:
W_hh(t+1) = W_hh(t) + CPPN(坐标, hidden(t), W_hh(t))
```

6个CPPN节点：
- w_ih: 生成输入→隐藏权重
- b_h: 生成隐藏层偏置
- w_hh: 生成隐藏→隐藏初始权重
- b_o: 生成输出层偏置
- w_ho: 生成隐藏→输出权重
- **delta_w**: 生成隐藏→隐藏权重更新规则（自适应核心）

#### 💻 代码示例

```python
from pytorch_neat.adaptive_net import AdaptiveNet
from pytorch_neat.activations import tanh_activation

# 定义神经元空间坐标
input_coords = [[-1, 0], [0, 0], [1, 0]]
hidden_coords = [[-0.5, 0.5], [0.5, 0.5]]  # 2个隐藏神经元
output_coords = [[-1, 1], [0, 1], [1, 1]]

# 从基因组创建网络
net = AdaptiveNet.create(
    genome,
    config,
    input_coords=input_coords,
    hidden_coords=hidden_coords,
    output_coords=output_coords,
    weight_threshold=0.2,
    activation=tanh_activation,
    batch_size=4,
    device='cuda:0'  # 可以使用GPU加速
)

# 重置网络
net.reset()

# 前向传播（自动更新循环权重）
for step in range(max_steps):
    outputs = net.activate(states)
    # 隐藏层循环权重已自动更新
```

#### 🔑 关键参数

**input_coords / hidden_coords / output_coords** (神经元坐标)
- 需要为三个层分别定义坐标
- 隐藏神经元的坐标特别重要，影响权重更新规则
- 建议: 隐藏层放在输入和输出之间的空间位置

**weight_threshold** (权重阈值)
- 作用: 控制所有权重矩阵的稀疏性
- 推荐: 0.1-0.3

**activation** (激活函数)
- 同时用于隐藏层和输出层
- 推荐: tanh（零中心，适合自适应）

#### ⚙️ 工作原理

1. **初始化**: 使用5个CPPN生成所有初始权重矩阵
   ```python
   W_ih = CPPN_ih(坐标, 0, 0, 0)
   W_hh = CPPN_hh(坐标, 0, 0, 0)
   # ... 其他权重
   ```

2. **前向传播**: 
   ```python
   # 更新隐藏层
   hidden(t) = activation(
       W_ih @ input(t) +
       W_hh(t) @ hidden(t-1) +
       bias_h)
   
   # 更新输出层
   output(t) = activation(
       W_ho @ hidden(t) +
       bias_o)
   ```

3. **权重更新**: 更新隐藏层循环权重
   ```python
   delta_w = CPPN_delta(
       坐标, hidden(t), hidden(t), W_hh(t))
   W_hh(t+1) = W_hh(t) + delta_w
   ```

#### ✅ 优点

- **最强能力**: 结合隐藏层、循环连接和自适应
- **长期记忆**: 隐藏层状态 + 权重变化
- **复杂推理**: 隐藏层可以进行复杂计算
- **元学习**: 可以"学会学习"
- **灵活性**: 适应各种复杂任务

#### ❌ 缺点

- **训练最慢**: 6个CPPN，计算量大
- **调参困难**: 参数多，难以优化
- **容易过拟合**: 能力强，需要大量数据
- **内存占用大**: 需要存储多个权重矩阵
- **难以调试**: 行为复杂，不易理解

#### 🎯 适用场景

- ✅ 复杂的记忆任务（长序列、多步推理）
- ✅ 非平稳环境（规则会变化）
- ✅ 元学习研究
- ✅ 多智能体协作
- ✅ 持续学习
- ⚠️ 不推荐用于简单任务（过度设计）

## 与NEAT基因组的关系

### 🧬 基因组编码方式

所有三种网络都使用NEAT基因组进化，但编码方式不同：

#### RecurrentNet - 直接编码

```
基因组 → 网络
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
节点基因 → 神经元（隐藏、输出）
连接基因 → 权重矩阵元素

示例:
节点1: bias=0.5, response=1.0
节点2: bias=-0.3, response=2.0
连接(0→1): weight=1.2
连接(0→2): weight=-0.8
```

**特点**:
- 每个权重都是一个可进化的参数
- 拓扑和权重同时进化
- 搜索空间大，但直观

#### AdaptiveLinearNet - CPPN间接编码

```
基因组 → CPPN → 权重
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
基因组定义CPPN的拓扑和权重
CPPN接收坐标等输入，生成权重更新规则

示例:
基因组编码: 10个节点, 15个连接
生成的CPPN可以产生: 100个网络权重

CPPN输入: [x_in, y_in, x_out, y_out, pre, post, w]
CPPN输出: [delta_w]
```

**特点**:
- 间接编码，参数效率高
- 可以利用几何规律性
- 搜索空间小，但需要合理的坐标设计

#### AdaptiveNet - 多CPPN间接编码

```
基因组 → 6个CPPN → 多个权重矩阵
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
一个基因组同时编码6个CPPN:
- CPPN1: 生成W_ih
- CPPN2: 生成b_h
- CPPN3: 生成W_hh初始值
- CPPN4: 生成b_o
- CPPN5: 生成W_ho
- CPPN6: 生成W_hh更新规则

示例:
基因组: 输入7个节点, 输出6个节点
create_cppn()自动分离成6个输出节点
```

**特点**:
- 最复杂的编码方式
- 共享基因组，协同进化
- 参数效率高，但训练慢

### 🔄 进化过程

所有网络都使用相同的NEAT进化算法：

```python
# 1. 评估种群
for genome in population:
    if net_type == 'recurrent':
        net = RecurrentNet.create(genome, config)
    elif net_type == 'adaptive_linear':
        net = AdaptiveLinearNet.create(genome, config, coords...)
    elif net_type == 'adaptive':
        net = AdaptiveNet.create(genome, config, coords...)
    
    fitness = evaluate(net, env)
    genome.fitness = fitness

# 2. 选择
parents = select_best(population)

# 3. 繁殖（交叉、变异）
offspring = reproduce(parents)

# 4. 物种划分
speciate(offspring)

# 5. 下一代
population = offspring
```

### 📊 基因组复杂度对比

| 网络类型 | 典型节点数 | 典型连接数 | 参数量 | 进化代数 |
|---------|-----------|-----------|-------|---------|
| RecurrentNet | 5-20 | 10-50 | 直接编码 | 100-500 |
| AdaptiveLinearNet | 3-10 | 5-20 | 间接编码 | 500-2000 |
| AdaptiveNet | 5-15 | 10-40 | 间接编码 | 1000-5000 |

**说明**:
- 节点数: CPPN的内部节点数（不是网络的神经元数）
- 连接数: CPPN的内部连接数
- 参数量: 直接编码=连接数，间接编码≈节点数×连接数
- 进化代数: 典型任务需要的代数
